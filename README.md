# 병뚜껑 게임

병뚜껑을 손가락으로 튕겨서 선에 맞추는 게임을 아시나요?

그 게임을 만들었습니다. 

병뚜껑을 마우스 드래그로 튕겨 선택지를 지나치면 선택지에 따라 병뚜껑 상태도 변하고, 거리순으로 랭킹을 세우는 게임입니다.

이제 병뚜껑을 날려보세요!
